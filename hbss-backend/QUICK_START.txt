========================================
HBSS LiveChat Backend - Quick Start
========================================

STEP 1: Open a NEW terminal/command prompt

STEP 2: Navigate to backend folder:
   cd enigma-forge-ui-main/hbss-backend

STEP 3: Run the start script:

   WINDOWS:
   start.bat

   MAC/LINUX:
   chmod +x start.sh
   ./start.sh

   OR MANUALLY:
   python -m venv venv
   venv\Scripts\activate (Windows)
   source venv/bin/activate (Mac/Linux)
   pip install -r requirements.txt
   python main.py

========================================
Troubleshooting:
========================================

Error: "python not found"
Solution: Install Python 3.9+ from python.org

Error: "Port 8000 already in use"
Solution: Close other programs using port 8000

Error: "Module not found"
Solution: Run: pip install -r requirements.txt
